## Описание

1. Управление сетью

## Примечание по сетевым настройкам

| Параметр | В задании | Реализовано | Причина |
|----------|-----------|-------------|---------|
| IP-адрес | 192.168.1.100/24 | 192.168.0.100/24 | VirtualBox NAT не поддерживает сеть 192.168.1.0/24 |
| Шлюз | 192.168.1.1 | 192.168.0.1 | Дефолтный шлюз VirtualBox |

### Итоговая конфигурация сети:
IP-адрес: 192.168.0.100/24
Шлюз: 192.168.0.1
DNS: 8.8.8.8
Hostname: mephi-2026.domain.local


Все сервисы (nginx, SSH) работают корректно, связность подтверждена.
